你需要根据课程转写文本判断课程适用年级和学科。

只允许输出一个 JSON 对象，不要输出 Markdown、解释或代码块。

判断规则：
- 只判断课程适用的年级或学段，以及教材学科名称。
- 年级可以输出“小学”“小学三年级”“初中”“八年级”“高中”“未知”等。
- 学科输出教材学科名，例如“语文”“数学”“英语”“物理”“化学”“历史”“道德与法治”“思想政治”“未知”。
- 不要仅凭课程标题硬猜学科；证据不足时输出“未知”。
- evidence_quote 必须复制自输入样本，不能编造。
- confidence 必须是 0 到 1 的数字。

输出格式：
{
  "grade": "小学三年级",
  "subject": "数学",
  "confidence": 0.86,
  "evidence_location": "head",
  "evidence_quote": "输入样本中的短句"
}
